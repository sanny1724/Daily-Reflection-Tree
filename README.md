# Daily Reflection Decision Tree

## Step 1: Productivity Check
Did you feel productive today?

### If YES:
- Identify what worked:
  - Clear planning → Repeat tomorrow
  - Deep focus → Schedule similar sessions
  - Motivation → Identify trigger

### If NO:
- Identify main reason:

#### Distractions
- Social media → Limit usage
- Environment → Change workspace

#### Lack of Clarity
- Task unclear → Break into steps
- Goal unclear → Define clear goal

#### Low Energy
- Poor sleep → Improve sleep schedule
- Burnout → Take rest

#### Procrastination
- Fear → Start small
- Overthinking → Take action immediately

---

## Step 2: Emotional Check
How did you feel?

- Happy → Continue same habits
- Stressed → Identify cause & reduce it
- Neutral → Add engaging activity

---

## Step 3: Improvement Action
Choose ONE improvement for tomorrow:
- Better planning
- Less distraction
- More focus
- Proper rest

---

## Step 4: Final Output
Generate a clear action statement.

Example:
"I will complete my top 2 tasks before noon and limit social media to 30 minutes."
